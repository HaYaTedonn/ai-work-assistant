# 業務アシスタントAI（デモ）

メール作成・議事録の要約・文章校正・キャッチコピーなど、**定型業務を助けるAIチャット**のデモです。

🔗 **ライブデモ:** https://hayatedonn.github.io/ai-work-assistant/

## 主な機能
- チャットUI（吹き出し・入力中ローディング・履歴保存）
- クイックアクション：**メール作成 / 議事録の要約 / 文章の校正 / キャッチコピー / 箇条書き整形**
- **デモモード（APIキー不要）**：簡易ロジックで“それっぽい”応答を生成。商談のライブでも止まらず動きます
- **実APIモード（任意）**：設定から OpenAI または Anthropic の APIキーを入れると、実際のAIが応答します

## 動作モードについて
- 既定は **デモモード**。APIキーなしで、すぐに動作を体験できます（応答は簡易ロジックによる擬似生成）。
- 設定で API プロバイダとキーを登録すると、本物のLLM応答に切り替わります。失敗時は自動でデモ応答にフォールバックします。
- 🔒 APIキーは**このブラウザ（localStorage）にのみ保存**され、送信先は各社APIだけです。サーバーには送信しません。

## 技術ポイント
- バニラ JavaScript + CSS、外部CDN・フレームワークなし
- localStorage による会話履歴・設定の保存
- OpenAI / Anthropic 両対応の API 呼び出し（ブラウザから直接、フォールバック付き）

## ファイル構成
- `index.html` … 単一ファイルで完結（CSS/JS内包）

## 補足
これは提案用デモです。デモモードの応答は簡易ロジックであり、実際の生成AIの品質とは異なります。
